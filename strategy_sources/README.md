# Strategy Sources

Place local strategy source documents here when you want them stored with the project, for example:

- `The Options Cartel Blueprint.pdf`
- `Options Blueprint Video Transcript.txt`
- focus-list examples or setup screenshots

The active Claude chart-triage context is the compressed prompt in `prompts/STRATEGY_MASTER.md`; the scanner does not permanently hardcode local Desktop paths. Use `--strategy-source-dir` or `TRADING_STRATEGY_SOURCE_DIR` for a different source directory if future extraction/summarization is added.
